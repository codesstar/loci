# 08-archive — Archive

Everything completed, expired, or no longer active lives here.

## Structure
```
tasks/      → Completed task batches
planning/   → Past plans and reviews
projects/   → Finished project notes
misc/       → Everything else
```

## Rules
- Archive, never delete
- Preserve original frontmatter
- Add `archived: YYYY-MM-DD` to frontmatter when archiving
- Only read this folder when looking up history (Layer 3)
