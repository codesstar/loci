# create-loci

Set up a [Loci](https://github.com/codesstar/loci) brain — Memory Palace for AI.

## Quick Start

```bash
npx create-loci
```

That's it. Works on Mac, Linux, and Windows.

## Options

```bash
npx create-loci [target-dir]   # Default: ~/loci
npx create-loci --web          # Force browser-based setup
npx create-loci --cli          # Force terminal setup (Mac/Linux)
```

## What it does

1. Downloads the Loci template (via `git clone` or direct download)
2. Runs the interactive setup wizard
3. Configures your AI tools to use the brain

## Requirements

- Node.js 18+
- That's it. Git is optional.
