The Problem

Every time you start a new AI conversation, it starts from zero.

  • ChatGPT memories — a flat list of disconnected facts. No structure, no context.
  • Custom GPTs — static. They don't grow with you.
  • .cursorrules — instructions without memory, identity, or orchestration.
You end up repeating yourself. Re-explaining your goals. The AI never truly knows you.

The Solution

A memory palace that loads automatically and grows with every conversation.

🧠

Three-Layer Context

L1 loads every conversation. L2 on demand. L3 stores deep history. Your AI stays fast.

⚗️

Distillation

Never saves raw transcripts. Extracts insights, decisions, and tasks into structured files.

👤

Identity Modeling

Your AI knows your goals, values, and working style. It evolves as you do.

⚡

Synapse System

Connect any folder. Info flows between brain and projects like synapses between neurons.

🔒

100% Local

All Markdown, all on your machine. No database, no server, no account.

📊

Dashboard

Pixel-art visualization of your goals, tasks, inbox, and project status.

How It Works

One brain, many projects. Each connects via /loci-link and communicates through the Synapse system.

Slash Commands

/loci-linkConnect folder + auto-scan
/loci-settingsConfigure sync
/loci-brain-settingsPrivacy & routing
/loci-syncPush + pull
/loci-scanRe-scan project

Quick Start

30 seconds. No setup script needed.

Terminal
# Clone the repository git clone https://github.com/codesstar/loci.git my-brain cd my-brain # Start your brain claude "help me set up my brain"

Loci detects it's your first time and walks you through setup. In 2 minutes, your brain is ready.
